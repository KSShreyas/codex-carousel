# Maestro web UI flows for Codex Carousel

These flows validate core click paths for `http://127.0.0.1:3000`.

## Install Maestro

- macOS/Linux:
  - `curl -fsSL "https://get.maestro.mobile.dev" | bash`
- Windows:
  - Follow the official install instructions: https://maestro.mobile.dev/getting-started/installing-maestro/windows

> Note: Maestro web support may be beta depending on your local browser/runtime setup.

## Run

```bash
npm run maestro:check
npm run maestro:test
```

If CLI is missing, `npm run maestro:check` prints install guidance.
