# User Guide

This guide is written for normal daily use.

## 1) Start the app

1. Run `npm install`
2. Run `npm run dev`
3. Open `http://127.0.0.1:3000`

## 2) If you see Setup Required

1. Click **Set Up Codex**.
2. Click **Scan for Codex**.
3. Confirm the **Codex data folder**.
4. Confirm the **Codex app path/launch command**.
5. Click **Save Setup**.

If setup is incomplete, account switching stays blocked for safety.

## 3) Add your first account

1. Click **Add Account**.
2. Click **Open Codex Login**.
3. Sign in to your account in Codex.
4. Return and click **I Logged In**.
5. Enter **Account Name** and choose **Plan**.
6. Click **Save This Account**.

## 4) Add another account

Repeat the same **Add Account** steps for your next account.

## 5) Switch accounts safely

1. Click **Switch** on the target account.
2. Review **Safety Check** results.
3. Confirm: **I understand Codex should be closed before switching.**
4. Click **Switch Account**.

If Codex is open, close it first and re-run Safety Check.

## 6) Open Codex

Click **Open Codex** in the top bar.

If it does nothing, check your app path in **Advanced Settings**.

## 7) Update usage manually

1. Click **Update Usage** on an account.
2. Set usage values.
3. Save.

This helps improve recommendation quality.

## 8) Recommendations

Recommendations suggest whether to stay or switch.

Recommendations never switch for you.

## 9) Advanced Settings and Diagnostics

Use **Advanced Settings** for:
- Codex data folder
- Codex app path
- switch safety toggles
- Diagnostics status and issues
- event log
