# User Guide (V1.0)

## 1) Start
1. `npm install`
2. `npm run dev`
3. Open `http://127.0.0.1:3000`

## 2) Capture profiles
Use **Capture current login** after signing into Codex with each desired account.

## 3) Record manual usage snapshot
Update 5H / weekly / credits status from observed usage signals.

## 4) Read recommendation
Recommendations are advisory only and never auto-switch.

## 5) Dry-run switch
Run dry-run first to inspect warning state and file plans.

## 6) Real switch
Enable local switching in settings and check confirmation before switching.

## 7) Launch + verify
Launch Codex and manually verify identity.

## 8) Use doctor + ledger
Use doctor for health checks and ledger for switch history / rollback visibility.
